#  Portfolio

A futuristic, anti-gravity themed personal portfolio website for Aman Ali, Software Engineer.

## Features
- **Hero Section**: Floating particles, magnetic buttons, and glitch text effects.
- **About Me**: Glassmorphism cards with hovering animations.
- **Projects**: 3D tilt effects on project cards.
- **Skills**: Orbiting skill bubbles.
- **Contact**: Neon-styled contact form.

## Technologies
- HTML5
- CSS3 (Variables, Keyframes, Flexbox/Grid)
- JavaScript (Canvas API, DOM Manipulation)

## Setup
1. Clone the repository.
2. Open `index.html` in your browser.
